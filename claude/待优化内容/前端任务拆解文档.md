## 五、后端API需求汇总

如果需要后端API支持，需要新增或扩展以下接口：

| 功能 | 需新增/扩展API | 说明 |
|------|---------------|------|
| 知识点状态 | 扩展 `GET /knowledge-points` | 增加掌握状态字段 |
| 同步数据 | 扩展 `GET /audio/:id/sync` | 文字时间戳、PPT切换点 |
| 笔迹数据 | 新建 `GET /audio/:id/handwriting` | 笔迹坐标数据 |
| 复习计划 | 扩展 `GET /review/calendar` | 三轮复习时间轴 |
| 复习进度 | 扩展 `GET /review/progress` | 各轮复习进度 |
