### 7.2 技术要点

**知识图谱D3交互**:
- 使用 `d3.forceSimulation` 实现力导向布局
- 节点点击事件：`onClick={() => onNavigate(AppView.TIME_MACHINE, node.id)}`
- 缩放功能：`d3.zoom()`

**时光机同步**:
- 音频-文字同步：维护 `currentTime` 状态
- 点击文字跳转：`audioRef.current.currentTime = timestamp`
- PPT翻页：`const activeIndex = slides.findIndex(s => time >= s.timeStart)`

**Canvas笔迹回放**:
- 解析笔迹数据：`[{x, y, timestamp, color}]`
- 动画循环：`requestAnimationFrame`
- 时间轴对齐：`const progress = currentTime / duration`
