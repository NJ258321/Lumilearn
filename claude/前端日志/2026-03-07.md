## 发现并修复的问题

### 1. TimeMachine.tsx - 中文全角引号错误
- **问题描述**：第 549-561 行使用了中文全角引号 `"` 而不是英文半角引号 `"`，导致编译错误
- **错误信息**：`Expected "{" but found "”"`
- **修复方案**：使用 Node.js 脚本将所有全角引号替换为半角引号

### 2. Recorder.tsx - 重复声明变量
- **问题描述**：`filteredCourses` 变量被声明了两次（第 95 行和第 441 行），导致 TypeScript 编译错误
- **错误信息**：`The symbol "filteredCourses" has already been declared`
- **修复方案**：删除第一个未使用的声明，保留使用 MOCK_COURSES 的版本

### 3. constants.ts - 枚举值大小写不匹配
- **问题描述**：MOCK_COURSES 中使用了小写的 status 和 type 值（如 `'reviewing'`, `'major'`），但类型定义要求大写
- **错误信息**：`Type '"reviewing"' is not assignable to type 'CourseStatus'`
- **修复方案**：将所有枚举值改为大写
  - `'reviewing'` → `'REVIEWING'`
  - `'studying'` → `'STUDYING'`
  - `'archived'` → `'ARCHIVED'`
  - `'major'` → `'PROFESSIONAL'`
  - `'elective'` → `'ELECTIVE'`

### 4. types.ts & src/types/api.ts - Course 类型缺少属性
- **问题描述**：`Course` 接口缺少 `lastReview` 和 `semester` 属性
- **修复方案**：在类型定义中添加可选的额外属性

### 5. src/types/index.ts - 类型导出问题
- **问题描述**：API 文件导入路径 `'../types'` 无法找到模块
- **修复方案**：创建 `src/types/index.ts` 重新导出类型

### 6. src/vite-env.d.ts - Vite 环境变量类型缺失
- **问题描述**：`import.meta.env` 类型未定义
- **修复方案**：创建 `src/vite-env.d.ts` 添加类型声明

### 7. pages/Analysis.tsx - 类型比较错误
- **问题描述**：`course.type === 'major'` 与类型定义不匹配
- **修复方案**：改为 `course.type === 'PROFESSIONAL'`

### 8. pages/CourseDetailReview.tsx - 未定义变量引用
- **问题描述**：使用了未定义的 `record` 变量
- **修复方案**：改为使用 `studyRecords[0]` 并添加条件判断

### 9. pages/CourseDetailStudy.tsx - 不支持的属性
- **问题描述**：`CreateKnowledgePointBody` 类型不支持 `description` 属性
- **修复方案**：移除该属性

### 10. pages/Dashboard.tsx - 参数名称错误
- **问题描述**：`getRecentlyReviewed` 使用了不存在的 `limit` 参数
- **修复方案**：改为 `pageSize`

### 11. pages/KnowledgeManager.tsx - 不支持的属性
- **问题描述**：同样存在 `description` 属性问题
- **修复方案**：移除该属性

### 12. pages/Recorder.tsx - 多个类型问题
- 修复 File 类型断言
- 修复 MOCK_COURSES 导入路径
- 修复课程类型比较

### 13. src/api/upload.ts - 类型参数问题
- **问题描述**：`validateFileType` 函数类型参数包含不必要的 `'unknown'`
- **修复方案**：调整类型定义

### 14. src/components/FilePreview/FilePreviewModal.tsx - 导入方式错误
- **问题描述**：`DocumentViewer` 使用了命名导入但实际是默认导出
- **修复方案**：改为默认导入

### 15. src/components/ImageUploader.tsx - File 类型问题
- **问题描述**：`files` 变量的类型推断为 `unknown[]`
- **修复方案**：添加类型断言 `as File[]`

### 16. src/components/StudySuggestions.tsx - null 检查和类型问题
- **问题描述**：`data.studyPlan` 可能为 null，tasks 类型推断为 unknown
- **修复方案**：添加 null 检查和类型断言

### 17. src/components/TimeMarker.tsx - 缺少类型定义
