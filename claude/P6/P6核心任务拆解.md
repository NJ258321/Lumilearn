## 二、任务拆解详情

### P6.1: 题库数据模型扩展

**任务ID**: P6.1
**文件**: 扩展 `code/backend/prisma/schema.prisma`
**输出**: Question 模型及相关表

**数据库模型**:
```prisma
// ==================== 题库系统 (P6) ====================

// 题目模型
model Question {
  id              String   @id @default(uuid())
  courseId        String
  chapterId       String?
  knowledgePointId String?

  // 题目内容
  type            String   // SINGLE_CHOICE | MULTIPLE_CHOICE | TRUE_FALSE | SHORT_ANSWER | ESSAY
  content         String   // 题目内容（支持Markdown）
  options         String?  // JSON数组：选项内容 [{ "A": "..." }, { "B": "..." }]
  answer          String   // 答案（JSON格式）
  explanation     String?  // 解析

  // 题目元数据
  difficulty      Int      @default(5) // 1-10 难度
  score           Int      @default(10) // 分值
  timeLimit       Int?     // 时间限制（秒）
  tags            String?  // 标签（JSON数组）
  source          String?  // 来源（真题/模拟题/练习题）

  // 统计信息
  usedCount       Int      @default(0) // 使用次数
  correctRate     Float    @default(0) // 正确率

  // 状态
  status          String   @default("ACTIVE") // ACTIVE | INACTIVE | REVIEW

  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt

  // 关联
  course          Course   @relation(fields: [courseId], references: [id], onDelete: Cascade)
  chapter         Chapter? @relation(fields: [chapterId], references: [id], onDelete: SetNull)
  knowledgePoint  KnowledgePoint? @relation(fields: [knowledgePointId], references: [id], onDelete: SetNull)

  // 答题记录
  answerRecords   AnswerRecord[]

  @@index([courseId])
  @@index([chapterId])
  @@index([knowledgePointId])
  @@index([type])
  @@index([difficulty])
  @@index([status])
}

// 答题记录
model AnswerRecord {
  id              String   @id @default(uuid())
  userId          String
  questionId      String
  examSessionId   String?

  // 答题内容
  userAnswer      String   // 用户答案
  isCorrect       Boolean  // 是否正确

  // 答题时间
  timeSpent       Int?     // 花费时间（秒）
  answeredAt      DateTime @default(now())

  // AI 分析
  aiAnalysis      String?  // AI 分析结果
  suggestedReview String?  // 建议复习内容

  // 关联
  user            User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  question        Question @relation(fields: [questionId], references: [id], onDelete: Cascade)

  @@index([userId])
  @@index([questionId])
  @@index([examSessionId])
  @@index([answeredAt])
}

// 考试/练习会话
model ExamSession {
  id              String   @id @default(uuid())
  userId          String
  courseId        String?

  // 会话信息
  type            String   // MOCK_EXAM | PRACTICE | RANDOM_TEST | AI_GENERATED
  title           String
  description     String?

  // 试卷信息
  questionIds     String?  // JSON数组：题目ID列表
  totalQuestions  Int      // 总题数
  totalScore      Int?     // 总分

  // 答题情况
  answeredCount   Int      @default(0)
  correctCount    Int      @default(0)
  score           Float?   // 得分

  // 时间
  startedAt       DateTime @default(now())
  completedAt     DateTime?

  // 状态
  status          String   @default("IN_PROGRESS") // IN_PROGRESS | COMPLETED | ABANDONED

  // 关联
  user            User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  course          Course?  @relation(fields: [courseId], references: [id], onDelete: SetNull)

  // 答题记录
  answerRecords   AnswerRecord[]

  @@index([userId])
  @@index([courseId])
  @@index([status])
  @@index([startedAt])
}
```

**预估时间**: 2小时

### P6.2: 题目 CRUD API

**任务ID**: P6.2
**文件**: 新建 `code/backend/src/routes/questions.ts`
**输出**: 题目管理 API

**API 接口详情**:

#### 2.1 获取题目列表
`GET /api/questions`

**查询参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| courseId | uuid | 课程ID筛选 |
| chapterId | uuid | 章节ID筛选 |
| type | string | 题目类型筛选 |
| difficulty | number | 难度筛选 |
| page | number | 页码 |
| pageSize | number | 每页数量 |
| search | string | 搜索关键词 |

**响应格式**:
```json
{
  "success": true,
  "data": {
    "questions": [...],
    "pagination": { "page": 1, "pageSize": 20, "total": 100 }
  }
}
```

#### 2.2 获取题目详情
`GET /api/questions/:id`

#### 2.3 创建题目
`POST /api/questions`

**请求体**:
```json
{
  "courseId": "uuid",
  "chapterId": "uuid",
  "knowledgePointId": "uuid",
  "type": "SINGLE_CHOICE",
  "content": "题目内容",
  "options": [{"A": "选项1"}, {"B": "选项2"}],
  "answer": {"correct": "A"},
  "explanation": "解析内容",
  "difficulty": 5,
  "score": 10,
  "tags": ["基础", "重要"],
  "source": "真题"
}
```

#### 2.4 更新题目
`PUT /api/questions/:id`

#### 2.5 删除题目
`DELETE /api/questions/:id`

#### 2.6 批量导入题目
`POST /api/questions/import`

**请求体**:
```json
{
  "questions": [
    { "type": "SINGLE_CHOICE", "content": "...", ... },
    ...
  ]
}
```

#### 2.7 导出题目
`GET /api/questions/export`

**查询参数**:
| 参数 | 说明 |
|------|------|
| courseId | 课程ID |
| format | 导出格式 (json/csv) |

**预估时间**: 4小时

### P6.3: 智能组卷算法

**任务ID**: P6.3
**文件**: 新建 `code/backend/src/routes/exams.ts`，新建 `code/backend/src/services/exam-generator.ts`
**输出**: 智能组卷 API

**API 接口详情**:

#### 3.1 AI 智能组卷
`POST /api/exams/generate`

**请求体**:
```json
{
  "courseId": "uuid",
  "title": "高等数学期中测试",
  "description": "涵盖第1-5章内容",
  "config": {
    "totalQuestions": 20,
    "totalScore": 100,
    "difficulty": {
      "easy": 0.3,
      "medium": 0.5,
      "hard": 0.2
    },
    "typeDistribution": {
      "SINGLE_CHOICE": 10,
      "MULTIPLE_CHOICE": 5,
      "TRUE_FALSE": 5
    },
    "knowledgePointCoverage": 0.8, // 知识点覆盖率
    "includeOldMistakes": true, // 是否包含历史错题相关
    "timeLimit": 90 // 分钟
  }
}
```

**响应格式**:
```json
{
  "success": true,
  "data": {
    "examSessionId": "uuid",
    "title": "高等数学期中测试",
    "questions": [
      {
        "id": "uuid",
        "type": "SINGLE_CHOICE",
        "content": "题目内容",
        "options": [...],
        "difficulty": 5,
        "score": 5,
        "knowledgePointId": "uuid"
      }
    ],
    "totalQuestions": 20,
    "totalScore": 100,
    "timeLimit": 90,
    "generatedAt": "2026-03-06T00:00:00Z"
  }
}
```

#### 3.2 按条件组卷
`POST /api/exams/generate-by-filters`

**请求体**:
```json
{
  "courseId": "uuid",
  "chapterIds": ["uuid1", "uuid2"],
  "knowledgePointIds": ["uuid1", "uuid2"],
  "types": ["SINGLE_CHOICE", "TRUE_FALSE"],
  "difficultyRange": [3, 7],
  "questionCount": 15,
  "randomOrder": true
}
```

#### 3.3 组卷策略配置
| 策略 | 说明 |
|------|------|
| random | 随机抽取 |
| difficulty_based | 按难度比例 |
| knowledge_coverage | 知识点覆盖优先 |
| mistake_based | 错题相关优先 |
| hybrid | 混合策略 |

**预估时间**: 6小时
