### P6-FE-02: TypeScript 类型定义

**任务ID**: P6-FE-02
**文件**: `src/types/api.ts`
**输出**: 添加 P6 相关 TypeScript 类型

**类型定义**:
```typescript
// 题目类型
export type QuestionType = 'SINGLE_CHOICE' | 'MULTIPLE_CHOICE' | 'TRUE_FALSE' | 'SHORT_ANSWER' | 'ESSAY'
export type QuestionStatus = 'ACTIVE' | 'INACTIVE' | 'REVIEW'

export interface Question {
  id: string
  courseId: string
  chapterId?: string
  knowledgePointId?: string
  type: QuestionType
  content: string
  options?: Record<string, string>[]
  answer: Record<string, any>
  explanation?: string
  difficulty: number
  score: number
  timeLimit?: number
  tags?: string[]
  source?: string
  usedCount: number
  correctRate: number
  status: QuestionStatus
  createdAt: string
  updatedAt: string
  course?: Course
  chapter?: Chapter
  knowledgePoint?: KnowledgePoint
}

// 答题记录
export interface AnswerRecord {
  id: string
  userId: string
  questionId: string
  examSessionId?: string
  userAnswer: string
  isCorrect: boolean
  timeSpent?: number
  answeredAt: string
  aiAnalysis?: string
  suggestedReview?: string
  question?: Question
}

// 考试会话
export type ExamSessionType = 'MOCK_EXAM' | 'PRACTICE' | 'RANDOM_TEST' | 'AI_GENERATED'
export type ExamSessionStatus = 'IN_PROGRESS' | 'COMPLETED' | 'ABANDONED'

export interface ExamSession {
  id: string
  userId: string
  courseId?: string
  type: ExamSessionType
  title: string
  description?: string
  questionIds?: string[]
  totalQuestions: number
  totalScore?: number
  answeredCount: number
  correctCount: number
  score?: number
  startedAt: string
  completedAt?: string
  status: ExamSessionStatus
  course?: Course
  questions?: Question[]
}

// 题目筛选
export interface QuestionFilters {
  courseId?: string
  chapterId?: string
  type?: QuestionType
  difficulty?: number
  status?: QuestionStatus
  search?: string
  page?: number
  pageSize?: number
}

// 组卷配置
export interface ExamGenerateConfig {
  totalQuestions: number
  totalScore?: number
  difficulty: { easy: number; medium: number; hard: number }
  typeDistribution?: Record<QuestionType, number>
  knowledgePointCoverage?: number
  includeOldMistakes?: boolean
  timeLimit?: number
}

// 随机抽题请求
export interface RandomDrawRequest {
  courseId: string
  count: number
  excludeIds?: string[]
  filters?: {
    type?: QuestionType
    difficulty?: [number, number]
    knowledgePointId?: string
  }
}

// 挑战模式
export type ChallengeMode = 'speed' | 'accuracy' | 'endurance'

// 答题统计
export interface ExamStatistics {
  totalAnswered: number
  correctCount: number
  correctRate: number
  averageTime: number
  byType: Record<QuestionType, { total: number; correct: number; rate: number }>
  byDifficulty: Record<string, { total: number; correct: number; rate: number }>
  weakAreas: { knowledgePointId: string; name: string; correctRate: number }[]
  trend: { date: string; correctRate: number }[]
}

// 错题
export interface Mistake {
  questionId: string
  content: string
  options?: Record<string, string>[]
  userAnswer: string
  correctAnswer: string
  wrongCount: number
  lastAnsweredAt: string
  knowledgePointId?: string
  knowledgePointName?: string
}
```

### P6-FE-03: API 服务文件

**任务ID**: P6-FE-03
**文件**: `src/api/questions.ts`, `src/api/exams.ts`
**输出**: P6 API 服务函数

**questions.ts 函数**:
- `getQuestionList(filters: QuestionFilters)` - 获取题目列表
- `getQuestionDetail(id: string)` - 获取题目详情
- `createQuestion(data: Partial<Question>)` - 创建题目
- `updateQuestion(id: string, data: Partial<Question>)` - 更新题目
- `deleteQuestion(id: string)` - 删除题目
- `importQuestions(courseId: string, questions: Partial<Question>[])` - 批量导入
- `exportQuestions(courseId?: string, format?: 'json' | 'csv')` - 导出题目

**exams.ts 函数**:
- `generateExam(courseId: string, title: string, config: ExamGenerateConfig)` - AI组卷
- `generateExamByFilters(data: any)` - 条件组卷
- `randomDraw(request: RandomDrawRequest)` - 随机抽题
- `getDailyPractice(courseId?: string, difficulty?: number)` - 每日一练
- `startChallenge(courseId: string, mode: ChallengeMode, count?: number)` - 挑战模式
- `createSession(data: { type: ExamSessionType; courseId?: string; title: string; questionIds?: string[] })` - 创建会话
- `submitAnswer(sessionId: string, questionId: string, userAnswer: string, timeSpent?: number)` - 提交答案
- `submitExam(sessionId: string)` - 交卷
- `getAnswerRecords(filters?: { questionId?: string; isCorrect?: boolean; page?: number; pageSize?: number })` - 获取答题记录
- `getExamStatistics(courseId?: string)` - 获取答题统计
- `getMistakes(knowledgePointId?: string)` - 获取错题本
- `retryMistakes(knowledgePointId?: string, count?: number)` - 错题重做
- `getPersonalizedPractice(courseId: string, count?: number, difficulty?: number)` - 个性化练习
