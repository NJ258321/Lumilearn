### P4.1: 学习数据统计与可视化 API

**任务ID**: P4.1
**文件**: 新建 `code/backend/src/routes/statistics.ts`
**输出**: 学习数据统计接口

**功能说明**:
- 统计课程学习时长、完成度、知识点掌握情况
- 提供可视化所需的数据结构
- 支持多维度数据聚合

**API 接口详情**:

#### 2.1 获取课程学习概览
`GET /api/statistics/course/:courseId/overview`

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| courseId | string | 是 | 课程ID |
| timeRange | string | 否 | 时间范围: week/month/semester |

**响应格式**:
```json
{
  "success": true,
  "data": {
    "courseId": "uuid",
    "courseName": "高等数学",
    "totalStudyTime": 3600, // 总学习时长(秒)
    "studyDays": 15, // 学习天数
    "knowledgePointsCount": 50,
    "masteredCount": 30,
    "weakCount": 10,
    "completionRate": 0.6,
    "averageMasteryScore": 72,
    "studyTimeByDay": [
      { "date": "2026-03-01", "time": 1800 },
      { "date": "2026-03-02", "time": 2400 }
    ],
    "knowledgeMasteryDistribution": {
      "excellent": 10,

      "medium": 15,
      "weak": 5
    }
  }
}
```

**响应格式**:

### P4.3: 复习计划智能生成

**任务ID**: P4.3
**文件**: 新建 `code/backend/src/routes/review.ts`
**输出**: 智能复习计划接口

**功能说明**:
- 基于艾宾浩斯遗忘曲线生成复习计划
- 智能安排复习时间和内容
- 支持按知识点、按章节多种维度

`POST /api/review/generate-plan`

      {

        "items": [
          {
            "type": "review",

## 四、数据库扩展

本次需要新增的模型：

```prisma
// P4.2 - 学习进度记录
model LearningProgress {
  id              String   @id @default(uuid())
  userId          String
  courseId        String
  chapterId       String?
  knowledgePointId String?
  status          String   // NOT_STARTED | LEARNING | MASTERED | WEAK
  masteryScore    Int?
  studyCount     Int      @default(0)
  lastStudiedAt   DateTime?
  createdAt       DateTime @default(now())
  updatedAt       DateTime @updatedAt

  @@index([userId, courseId])
  @@index([knowledgePointId])
}

// P4.3 - 复习计划
model ReviewPlan {
  id              String   @id @default(uuid())
  userId          String
  courseId        String
  knowledgePointId String
  reviewDate      DateTime
  reviewCount     Int      @default(1)
  status          String   // PENDING | COMPLETED | SKIPPED
  completedAt     DateTime?
  difficulty      String?  // easy | medium | hard
  timeSpent       Int?     // 秒
  notes           String?
  createdAt       DateTime @default(now())

  @@unique([userId, knowledgePointId, reviewDate])
  @@index([userId, reviewDate])
}

// P4.3 - 复习里程碑
model Milestone {
  id              String   @id @default(uuid())
  userId          String
  courseId        String
  title           String
  description     String?
  targetDate      DateTime
  progress        Float    @default(0)
  status          String   // PENDING | IN_PROGRESS | COMPLETED
  completedAt     DateTime?
  createdAt       DateTime @default(now())

  @@index([userId, courseId])
}

// P4.5 - 学习评估记录
model LearningEvaluation {
  id              String   @id @default(uuid())
  userId          String
  courseId        String
  timeRange       String   // week | month | semester
  overallScore    Int
  grade           String
  dimensions      Json      // 各维度评分
  strengths       String[]
  weaknesses      String[]
  suggestions     String[]
  trend           String    // improving | stable | declining
  generatedAt     DateTime @default(now())

  @@unique([userId, courseId, timeRange, generatedAt])
  @@index([userId, courseId])
}
```
