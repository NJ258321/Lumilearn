## 五、API对照表

**后端提供的API**:

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/upload/audio` | POST | 上传音频 |
| `/api/upload/image` | POST | 上传图片 |
| `/api/upload/document` | POST | 上传文档 |
| `/api/upload/:filename` | GET | 获取/预览文件 |
| `/api/upload/:filename` | DELETE | 删除文件 |

**前端需要映射的URL**:
- 图片: `/api/upload/{filename}` → `<img src="...">` 或 `<iframe>`
- 音频: `/api/upload/{filename}` → `<audio src="...">`
- 视频: `/api/upload/{filename}` → `<video src="...">`
- 文档: `/api/upload/{filename}` → PDF预览或下载
