### P2-F3: 创建音频元数据展示组件

**任务ID**: P2-F3
**对应后端**: P2.1
**文件**: `code/lumilearn-ai/src/components/AudioMetadata.tsx` (新建)

**功能说明**:
- 展示音频文件的元数据信息
- 显示时长、格式、采样率、比特率等
- 显示关联的知识点数量
- 显示时间标记数量

**元数据字段**:
| 字段 | 说明 |
|------|------|
| duration | 时长（秒）|
| format | 音频格式 |
| bitrate | 比特率 |
| sampleRate | 采样率 |
| channels | 声道数 |
| size | 文件大小 |
| knowledgePoints | 关联知识点 |
| timeMarksCount | 时间标记数量 |

**预估时间**: 30分钟

### P2-F5: 创建VAD可视化组件

**任务ID**: P2-F5
