## 二、任务拆解详情

### P5.1: 用户认证与权限管理

**任务ID**: P5.1
**文件**: 新建 `code/backend/src/routes/auth.ts`，扩展 `schema.prisma`
**输出**: 用户认证 API

**功能说明**:
- 用户注册、登录、登出
- JWT token 认证
- 用户权限管理（普通用户/管理员）
- 密码加密与安全存储

**API 接口详情**:

#### 1.1 用户注册
`POST /api/auth/register`

**请求体**:
```json
{
  "username": "string",
  "email": "string",
  "password": "string",
  "displayName": "string"
}
```

**响应格式**:
```json
{
  "success": true,
  "data": {
    "id": "uuid",
    "username": "string",
    "email": "string",
    "displayName": "string",
    "role": "USER",
    "createdAt": "2026-03-06T00:00:00Z"
  },
  "token": "jwt_token"
}
```

#### 1.2 用户登录
`POST /api/auth/login`

**请求体**:
```json
{
  "email": "string",
  "password": "string"
}
```

#### 1.3 获取当前用户
`GET /api/auth/me`

#### 1.4 更新用户信息
`PUT /api/auth/profile`

#### 1.5 修改密码
`PUT /api/auth/password`

**数据库模型扩展**:
```prisma
model User {
  id            String   @id @default(uuid())
  username      String   @unique
  email         String   @unique
  passwordHash  String
  displayName   String?
  role          String   @default("USER") // USER | ADMIN
  preferences   Json?    // 用户偏好设置
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt

  // 关联
  studyRecords  StudyRecord[]
  reviewPlans   ReviewPlan[]
  milestones    Milestone[]

  @@index([email])
  @@index([username])
}
```

**预估时间**: 4小时

### P5.3: 学习提醒服务

**任务ID**: P5.3
**文件**: 新建 `code/backend/src/routes/reminder.ts`，新建 `code/backend/src/services/reminder.ts`
**输出**: 学习提醒 API

**功能说明**:
- 定时学习提醒
- 复习任务提醒
- 考试倒计时提醒
- 目标达成提醒

**API 接口详情**:

#### 3.1 设置学习提醒
`POST /api/reminders`

**请求体**:
```json
{
  "type": "study_time | review | exam | goal",
  "title": "string",
  "message": "string",
  "scheduledAt": "2026-03-06T09:00:00Z",
  "recurring": {
    "enabled": true,
    "pattern": "daily | weekly",
    "daysOfWeek": [1, 2, 3, 4, 5]
  },
  "courseId": "uuid"
}
```

**响应格式**:
```json
{
  "success": true,
  "data": {
    "id": "uuid",
    "type": "study_time",
    "title": "早晨学习提醒",
    "message": "该开始今天的学习了！",
    "scheduledAt": "2026-03-06T09:00:00Z",
    "recurring": {
      "enabled": true,
      "pattern": "daily"
    },
    "status": "active"
  }
}
```

#### 3.2 获取提醒列表
`GET /api/reminders`

#### 3.3 更新提醒
`PUT /api/reminders/:id`

#### 3.4 删除提醒
`DELETE /api/reminders/:id`

#### 3.5 获取今日提醒
`GET /api/reminders/today`

**响应格式**:
```json
{
  "success": true,
  "data": {
    "date": "2026-03-06",
    "reminders": [
      {
        "id": "uuid",
        "type": "study_time",
        "title": "早晨学习提醒",
        "message": "该开始今天的学习了！",
        "scheduledAt": "09:00",
        "status": "pending"
      }
    ]
  }
}
```

#### 3.6 标记提醒完成
`POST /api/reminders/:id/complete`

**数据库模型**:
```prisma
model Reminder {
  id          String   @id @default(uuid())
  userId      String
  type        String   // study_time | review | exam | goal
  title       String
  message     String?
  scheduledAt DateTime
  recurring   Json?    // { enabled: boolean, pattern: string, daysOfWeek: number[] }
  status      String   @default("pending") // pending | completed | cancelled
  completedAt DateTime?
  createdAt   DateTime @default(now())

  @@index([userId, scheduledAt])
  @@index([userId, status])
}
```

**预估时间**: 4小时

### P5.4: 数据导出与同步

**任务ID**: P5.4
**文件**: 新建 `code/backend/src/routes/export.ts`
**输出**: 数据导出同步 API

**功能说明**:
- 导出学习数据（JSON/CSV）
- 导出学习报告
- 数据备份
- 多设备同步

**API 接口详情**:

#### 4.1 导出学习数据
`POST /api/export/data`

**请求体**:
```json
{
  "format": "json | csv",
  "dataTypes": ["courses", "chapters", "knowledge_points", "study_records", "mistakes", "notes"],
  "timeRange": {
    "start": "2026-01-01",
    "end": "2026-03-06"
  }
}
```

**响应**: 返回文件下载

#### 4.2 导出学习报告
`POST /api/export/report`

**请求体**:
```json
{
  "courseId": "uuid",
  "format": "pdf | json",
  "include": ["progress", "statistics", "evaluation", "recommendations"]
}
```

**响应**: 返回文件下载

#### 4.3 获取同步状态
`GET /api/sync/status`

**响应格式**:
```json
{
  "success": true,
  "data": {
    "lastSyncAt": "2026-03-06T10:00:00Z",
    "syncStatus": "synced",
    "pendingChanges": 0,
    "devices": [
      {
        "deviceId": "uuid",
        "deviceName": "MacBook Pro",
        "lastActive": "2026-03-06T10:00:00Z"
      }
    ]
  }
}
```

#### 4.4 触发同步
`POST /api/sync/trigger`

#### 4.5 导入数据
`POST /api/import/data`

**请求体**:
```json
{
  "format": "json",
  "mergeStrategy": "replace | merge" // 替换或合并
}
```

## 四、数据库扩展

本次需要新增的模型：

```prisma
// P5.1 - 用户
model User {
  id            String   @id @default(uuid())
  username      String   @unique
  email         String   @unique
  passwordHash  String
  displayName   String?
  role          String   @default("USER")
  preferences   Json?
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt

  // 关联
  studyRecords  StudyRecord[]
  reviewPlans   ReviewPlan[]
  milestones    Milestone[]
  reminders     Reminder[]
  recommendations Recommendation[]
  exportRecords ExportRecord[]

  @@index([email])
  @@index([username])
}

// P5.3 - 提醒
model Reminder {
  id          String   @id @default(uuid())
  userId      String
  type        String
  title       String
  message     String?
  scheduledAt DateTime
  recurring   Json?
  status      String   @default("pending")
  completedAt DateTime?
  createdAt   DateTime @default(now())

  @@index([userId, scheduledAt])
  @@index([userId, status])
}

// P5.2 - 推荐记录
model Recommendation {
  id            String   @id @default(uuid())
  userId        String
  itemType      String   // knowledge_point | resource
  itemId        String
  reason        String?
  priority      String   // high | medium | low
  status        String   // pending | viewed | completed | skipped
  feedback      String?
  createdAt     DateTime @default(now())
  completedAt   DateTime?

  @@index([userId, status])
  @@index([userId, createdAt])
}

// P5.4 - 导出记录
model ExportRecord {
  id          String   @id @default(uuid())
  userId      String
  type        String   // data | report
  format      String   // json | csv | pdf
  status      String   @default("pending")
  fileUrl     String?
  createdAt   DateTime @default(now())
  completedAt DateTime?

  @@index([userId, createdAt])
}
```

## 七、风险与注意事项

| 风险 | 应对措施 |
|------|----------|
| 用户认证安全 | 使用 bcrypt 加密，JWT token 设置过期时间 |
