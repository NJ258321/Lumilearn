### 协作规则（请严格遵守）

1. **严谨性**：生成的代码必须包含 TypeScript 类型定义
2. **完整性**：每次修改 Schema 后，请提醒我运行 `npm run prisma:migrate`
3. **错误处理**：API 必须包含 try-catch 和标准错误响应
4. **拼写检查**：确保枚举值拼写正确！
   - 正确：`'STUDYING'`, `'PROFESSIONAL'`, `'MASTERED'`, `'NEED_REVIEW'`
   - 错误：`'STUDYING'`, `'PROFESSIONAL'`, `'MASTERED'`, `'NEED_REVIEW'`
5. **响应格式**：
   - 成功：`{ success: true, data: ... }`
   - 失败：`{ success: false, error: '...' }`
   - 验证失败：`{ success: false, error: 'Validation failed', details: [...] }`
6. **路由注册**：创建新路由后，需要在 `src/index.ts` 中注册
7. **API 文档**：每完成一个模块，AI 会更新 `claude/API接口文档.md`

### Prisma 错误码参考

| 错误码 | 说明 | HTTP状态 |
|--------|------|---------|
| P2002 | 唯一约束冲突（如 order 重复）| 400 |
| P2003 | 外键约束失败（关联记录不存在）| 404 |
| P2025 | 记录不存在 | 404 |
