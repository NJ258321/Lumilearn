---

### 2.2 P2.1: 音频元数据提取 API

**接口地址**: `GET /api/audio/:id/metadata`

**功能**:
- 根据学习记录 ID 获取音频元数据
- 返回时长、格式、文件大小等信息
- 关联的知识点和时间标记统计

**测试结果**: ✅ 通过

---

### 2.3 P2.2: 音频转写接口（预留）

**接口地址**: `POST /api/audio/:id/transcribe`

**功能**:
- 预留框架，支持后续集成语音识别服务
- 阿里云、讯飞、Google Speech-to-Text

**额外接口**: `GET /api/audio/transcribe/status` - 获取服务配置状态

**测试结果**: ✅ 通过

---

### 2.4 P2.3: 音频片段提取 API

**接口地址**: `GET /api/audio/:id/segment`

**功能**:
- 根据时间范围提取音频片段
- 使用 ffmpeg 进行处理
- 最大片段时长 5 分钟
- 流式返回，自动清理临时文件

**测试结果**: ✅ 通过

---

### 2.5 P2.4: 语音活动检测 API

**接口地址**: `POST /api/audio/:id/vad`

**功能**:
- 使用 ffmpeg silencedetect 检测语音活动
- 支持灵敏度、最小语音时长、最小静音间隔参数
- 自动降级处理（ffmpeg 不可用时）

**额外接口**: `GET /api/audio/vad/status` - 获取 VAD 服务状态

**真实音频测试**:
- 测试文件: `1.1 映射与函数.mp3` (23MB, 10分钟)
- 检测到 142 个语音片段
- 总语音时长: 681.1秒

**测试结果**: ✅ 通过
