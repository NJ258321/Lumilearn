## 四、API对照表

**后端提供的API**:

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/knowledge-points/:id/relations` | GET | 获取知识关系图 |
| `/api/knowledge-relations` | POST | 创建知识关系 |
| `/api/knowledge-relations/batch` | POST | 批量创建 |
| `/api/knowledge-relations/:id` | PUT | 更新知识关系 |
| `/api/knowledge-relations/:id` | DELETE | 删除知识关系 |
| `/api/ai/explain` | POST | 生成知识点解释 |
| `/api/ai/suggest` | POST | 生成学习建议 |
| `/api/ai/analyze-weak-points` | POST | 分析薄弱点 |
| `/api/ai/search-resources` | POST | 搜索外部资源 |
